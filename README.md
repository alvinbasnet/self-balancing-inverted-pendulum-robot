# Self-Balancing Inverted Pendulum System

## Overview
Designed and built a real-time inverted pendulum system
using Arduino-based control.

## System Features
- PID and Pole Placement controllers
- Joystick input for motion control
- LED indicators for stability feedback

## Hardware
- Arduino UNO
- DC motors
- IMU sensors
- LEDs

## Software
- Arduino C/C++
- Control algorithms

## Results
- Stable balancing achieved
- Successful controller switching

## Skills Demonstrated
- Control systems
- Sensor integration
- System troubleshooting
